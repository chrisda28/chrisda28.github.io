Predictive Credit Risk Model & Web Application

A group project done while at CWRU and sponsored by KeyBank. We analyzed 210,000 records of consumer lending data using Matplotlib, Seaborn, Pandas, and Sci-kit Learn libraries. Tested and refined logistic regression, random forest, and gradient boosting approaches to credit risk modeling using Python. We decided to use a logistic regression model to enhance explainability, and incorporated features covering credit history, loan purpose, debt-to-income ratio, and more. Finally, we created an interactive web application with Typescript/React to host the credit risk model.

  • User friendly interface with Typescript/React
  • Credit risk model trained from 210,000 records of consumer lending data
  • Working prototype of web app using logistic regression model

Dividend Empirical Analysis

An individual project done while at CWRU. This project analyzes firms that begin paying dividends after a period of not doing so. It investigates the market's reaction to these dividend initiation announcements of 551 firms, the factors influencing this reaction, and the characteristics of firms that decide to start paying dividends. I developed multiple and logistic regression models to determine the likelihood of a firm initating dividends while incorporating control and interaction effects.

  • See the full report of results in the GitHub repo below
  • Markets react positively in a statistically significant manner to dividend initiations.
  • SAS code tests for heteroscedasticity
  • Firm size and firm age are found to be the biggest determinants of firm's decision to intiate dividends.

Bank Stock Analysis

Python scripts make API call to get historical time series data for JPMorgan, Bank of America, Goldman Sachs, and Citigroup for the past 20 years. The data is formatted and analyzed with Pandas and used to make visualizations with Seaborn that are rendered in Flask. For JPMorgan specifically, using scikit-learn, the program entails a basic linear regression model to predict the daily return (price change as a percentage) for the next day.

  • Languages: Python, HTML, CSS
  • Calculate metrics: correlation, std dev, 20-day MA, linear regression.
  • Visualize metrics in a Flask web application with commentary.
  • Technologies: Flask, Pandas, Scikit-learn, AlphaVantage API, Seaborn, Bootstrap
